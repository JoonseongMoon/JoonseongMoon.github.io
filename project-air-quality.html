Overview

Why Trustworthiness Matters

Digital twins promise real-time mirroring of physical buildings, but how much can we trust their predictions? This research develops a systematic framework for auditing data quality and quantifying uncertainty in digital twin outputs for the CMU Intelligent Workplace.

The goal is to move beyond black-box predictions toward transparent, calibrated confidence intervals that building operators can actually rely on for decision-making.

A sample of the data quality scoring pipeline:

data_quality.py 
import pandas as pd
import numpy as np

def compute_quality_score(df, col):
    """Compute data quality score for a sensor column."""
    completeness = 1 - df[col].isna().mean()
    
    # Detect anomalies via IQR
    Q1, Q3 = df[col].quantile([0.25, 0.75])
    iqr = Q3 - Q1
    outlier_rate = ((df[col] < Q1 - 1.5*iqr) | 
                    (df[col] > Q3 + 1.5*iqr)).mean()
    
    # Weighted composite score
    score = 0.6 * completeness + 0.4 * (1 - outlier_rate)
    return round(score, 3)
